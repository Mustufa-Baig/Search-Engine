# Simple Search Engine 
a basic Python web-crawler that recursively scans webpages
## Installation
Pre-requisite:
* Python 3.12 or above
* BeautifulSoup
```
pip install beautifulsoup4
```
## Usage
1. Run `spider.py` and provide inputs, `Seed ->` and `Limit ->`
   * `Seed` is the initial Webpage 
   * `Limit` is the maximum number of webpages to be scanned
     
2. Run `search.py` and enter a search term/key-word
4. the webpages with the most occurances of said key-word will appear
